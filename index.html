h1 { }

Заголовок типа h1

p { }

текст расположенный внутри элемента p

#id { }

элемент с данным id

.class { }

элементы класса

* { }

все элементы страницы

R D { }

R - родительский элемент, а D — дочерний элемент, расположенный в контейнере родительского

R > D { }

стиль применяется только для D, если он дочерний для R

R + D { }

стиль применяется только для D, если он соседний для R

R ~ D { }

стиль применяется для D, если он расположен после R

[атрибут="<значение>"]

применяется ко всем тегам, которые содержат указанное значение атрибута

E[атрибут="<значение>"]

применяется только к определённым селекторам

[атрибут^="<значение>"]

стиль применяется ко всем элементам, у которых значение атрибута начинаются с указанного текста

E[атрибут^="<значение>"]

только к определенным селекторам у которых значение атрибута начинаются с указанного текста

[атрибут$="значение"]

стиль для элемента, если значение атрибута оканчивается указанным текстом

[атрибут*="значение"]

когда стиль следует применить к тегу с определённым атрибутом, когда частью его значения является некоторый текст
